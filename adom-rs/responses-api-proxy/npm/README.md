# @openai/adom-responses-api-proxy

<p align="center"><code>npm i -g @openai/adom-responses-api-proxy</code> to install <code>adom-responses-api-proxy</code></p>

This package distributes the prebuilt [Adom Responses API proxy binary](https://github.com/openai/adom/tree/main/adom-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/adom-responses-api-proxy.js --help
```

Refer to [`adom-rs/responses-api-proxy/README.md`](https://github.com/openai/adom/blob/main/adom-rs/responses-api-proxy/README.md) for detailed documentation.
